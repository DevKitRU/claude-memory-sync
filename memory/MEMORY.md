# Memory Index

Это шаблон. Индекс-файл загружается в контекст Claude автоматически при старте каждой сессии.

Держи его под ~150 строк — после этого записи обрезаются. Если становится густо, переноси устаревшее в `archive/YYYY-MM.md`.

Один файл — одна строчка: `- [file.md](file.md) — краткий хук для релевантности`.

## User

- (пример) [user_profile.md](user_profile.md) — роль, цели, чем занимаюсь

## Feedback

- (пример) [feedback_<topic>.md](feedback_<topic>.md) — правило работы

## Project

- (пример) [project_<name>.md](project_<name>.md) — активная инициатива

## Reference

- (пример) [reference_<system>.md](reference_<system>.md) — ссылки на внешние системы
